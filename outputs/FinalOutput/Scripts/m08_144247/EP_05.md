# 重生逆袭 - 第 5 集

林晚意【转身走向电梯】：周景明，你猜警察多久会到？

周景明【冲上前拦住电梯门】：等等！我还有价值！我知道陆子谦和谁在做海外洗钱！

林晚意【按下关门键】：太迟了。你的价值在背叛那一刻就归零了。

林晚意【透过电梯门缝看他】：去跟警察解释，你是怎么“碰巧”出现在赃物现场的。

林晚意【在电梯里拨通电话】：顾律师，周景明在滨江公寓三楼，涉嫌敲诈和入室盗窃。

陆子谦【在警车内疯狂拍打隔板】：周景明这个废物！连拖延时间都做不到！

陆子谦【突然压低声音对警察笑】：同志，我要举报林晚意非法获取商业机密。

陆子谦【靠回座椅闭眼】：她手里那份U盘...是雇黑客从我电脑偷的。

林晚意【在电梯里挂断电话】：陆子谦，垂死挣扎的样子真难看。

林晚意【快步走出公寓大堂】：顾律师，陆子谦在警车上反咬一口，说U盘是非法获取。

林晚意【坐进车内启动引擎】：立刻把画廊暗格的账本拿到手，那是唯一能钉死他的铁证。

林晚意【猛打方向盘】：顺便，把周景明“主动”提供的洗钱线索也送给警方。

林晚意【疾驰中瞥见后视镜】：有辆车在跟...陆家的人反应真快。

林晚意【猛踩油门甩尾入巷】：想截胡？也得看有没有那个本事。

林晚意【拨通顾知行电话】：我被跟踪了，你那边情况如何？

林晚意【听到电话那头嘈杂声眼神一凛】：暗格被破坏了？指纹拓印用不了？

林晚意【急刹掉头】：备用方案。用陆子谦母亲的声音解锁，我这里有合成音频。

林晚意【将音频文件发送出去】：三分钟内必须打开，否则账本会被远程销毁。

林晚意【瞥见跟踪车辆逼近】：看来得先解决尾巴。

林晚意【突然减速让后车追尾】：抱歉，你全责。现在，下车谈谈？

陆子谦【在警车内突然睁眼】：停车！我要举报林晚意雇凶杀人！

林晚意【推门下车走向后车】：雇凶杀人？陆子谦，你编故事的能力退步了。

林晚意【敲击对方车窗】：跟踪我这么久，是陆家派你来抢账本的？

林晚意【瞥见对方腰间鼓起】：带家伙了？正好，让警察看看陆家狗急跳墙的嘴脸。

林晚意【突然拉开车门夺过手机】：在给谁报信？陆老爷子还是他那个律师？

林晚意【将手机扔进路边水沟】：告诉陆家，账本十分钟后就会出现在经侦科桌上。

林晚意【转身回拨顾知行电话】：暗格打开了没有？合成音频只能维持五分钟。

林晚意【听到电话那头开锁声松口气】：拿到了？立刻扫描关键页发给我。

林晚意【挂断电话冷眼看向跟踪者】：现在，你可以回去告诉陆家——游戏结束了。
