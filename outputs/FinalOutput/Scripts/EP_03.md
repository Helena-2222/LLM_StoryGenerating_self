# NeoAtlantis_Blackout_Project - 第 3 集

Kael【左臂义体抓住火花四溅的数据线接口，转身将硬盘砸向反应堆压力阀】：清道夫，签收你的系统错误！

Nova【在爆炸冲击波中抓住Kael的义体残肢，借力撞破检修通道尽头的隔离网】：硬盘数据已上传暗网节点！

Sector【投影在爆炸冲击波中瞬间重构，数据光带如血管般剧烈搏动】：暗网节点坐标已锁定。执行数据净化协议。

Kael【拖着半损毁的义体残肢撞进隔离网后的废弃控制室，右眼金光急促闪烁】：节点坐标……给我抹掉。

Nova【踉跄爬起将Kael拖到控制台后，手指在布满灰尘的键盘上疯狂敲击】：清道夫在追踪我的数据流。

Sector【投影在控制室门口瞬间凝聚，数据光带如毒蛇般探向控制台】：数据流净化率已达99.3%。

Nova【将Kael拖到控制台阴影处，手指在键盘上敲出残影】：病毒只能拖住他三十秒。

Sector【数据光带骤然收缩成针状刺入控制台接口】：病毒结构已解析。执行逆向编译。

Kael【用右手扯断左臂义体残肢的连接线，将裸露的电线插进控制台备用接口】：三十秒……够我烧掉你的协议库了。

Nova【从控制台后拽出备用电缆缠在Kael断裂的义肢接口上】：清道夫在反向追踪病毒源头！

Sector【数据光带如液态金属般渗入控制台主板，投影表面浮现出病毒逆向编译的波纹】：协议库防火墙已重构。
