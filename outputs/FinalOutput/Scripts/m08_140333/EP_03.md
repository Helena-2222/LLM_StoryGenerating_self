# 重生逆袭 - 第 3 集

林晚意【抬手拦住电梯门】：等等。

林晚意【从手包取出U盘递给警察】：完整证据链，包括他们海外账户的流水。

林晚意【目送警察带走两人】：账目已经清空，该查内部了。

陆子谦【被警察按住时突然抬头】：晚意！画廊账目有问题...你最好亲自查查！

林晚意【眼神骤然锐利】：画廊账目？你果然还留了后手。

林晚意【快步走向电梯按钮】：警察先生，请稍等三分钟。

陆子谦【被按在墙上却露出诡异笑容】：画廊的备用账本...在苏婉儿公寓的保险箱里。

林晚意【按住电梯门对警察微笑】：抱歉，我需要确认一个细节。

林晚意【转向陆子谦压低声音】：保险箱密码是多少？

陆子谦【挣扎着露出嘲讽表情】：你猜？就像我猜不透你什么时候变得这么狠一样。

林晚意【从手包抽出钢笔抵在他颈侧】：最后一次机会。密码，或者我让记者报道你试图袭警。

林晚意【对警察点头致意】：他刚才试图抢夺证据，您看到了吧？

陆子谦【咬牙低吼】：0715...我们的订婚日。满意了？

林晚意【收回钢笔轻笑】：真讽刺。用纪念日藏赃款账本。

林晚意【退后一步让出通道】：警察先生，请务必查清画廊所有关联账户。

林晚意【快步走向消防通道】：备用账本...看来得赶在内部反应前拿到。

陆子谦【被押进电梯时突然回头】：晚意！U盘有备份...在画廊休息室绿植盆底！

林晚意【疾步冲下消防通道】：绿植盆底...他果然还藏了后手。

林晚意【冲出酒店后门拦车】：去云锦画廊，快。

陆子谦【在警车内突然低笑】：晚意...你猜我为什么故意说漏保险箱？

陆子谦【转向苏婉儿耳语】：账本早转移了...她拿到的会是惊喜。

林晚意【猛踩刹车转向】：师傅，改去滨江公寓。

林晚意【拨通顾知行电话】：画廊是陷阱，真账本在苏婉儿公寓。

林晚意【疾步冲进公寓电梯】：密码0715...他故意说漏得太明显了。

林晚意【踹开虚掩的房门】：果然被翻过了...保险箱是空的。

林晚意【蹲下检查地毯压痕】：有人比我先到...痕迹还是湿的。

林晚意【拨通顾知行电话】：账本被转移了，立刻查陆子谦母亲名下的保险柜。
